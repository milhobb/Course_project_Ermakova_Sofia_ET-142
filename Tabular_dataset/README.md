Выбранный кейс №56: Классификация эффективности когнитивно-поведенческой терапии при депрессии. Исходные данные: Данные пациентов: возраст, длительность симптомов, качество сна (шкала), уровень тревожности (HADS), количество стрессовых событий за последние 6 месяцев, тип депрессии (реактивная, эндогенная, сезонная — при обучении). Требуется: Построить модель классификации, предсказывающую успех терапии.
