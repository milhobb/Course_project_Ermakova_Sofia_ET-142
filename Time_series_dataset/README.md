Выбранный кейс №64: Раннее обнаружение перегрева подшипника по тренду температуры. 
Исходные данные: Многомерные температурные временные ряды с датчиков.
Требуется: Разработать модель прогноза аномалии на основе трендов и градиентов температуры. Цель — обнаружение неисправности за 10–30 минут до критического порога.
