Выбранный кейс №62: Диагностика подшипников с помощью гибридной CNN-MLP модели 
Исходные данные: Вибрационные сигналы с экспериментального  стенда, 5 классов состояний подшипников. Требуется: Построить гибридную модель: CNN для извлечения паттернов из спектрограмм, MLP для работы с числовым вектором из статистических признаков. Модель должна обеспечивать >95% точности.


Выбранный датасет: Case Western Reserve University (CWRU) Bearing Dats Center


Общая информация: название - Case Western Reserve University (CWRU) Bearing Dats Center (https://engineering.case.edu/bearingdatacenter/download-data-file). Источник - официальный сайт университета. Авторы -  исследовательская группа CWRU под руководством профессора Kenneth A.Loparo. Дата публикации - данные собирались в конце 1990-х - начле 2000-х годов. Лицензия: данные находятся в открытом доступе для академических и исследовательских целей.


Описание содержимого: тип задачи - диагностика подшипников с помощью гибридной CNN-MLP модели (классификация временных рядов), формат файлов - .mat(MATLAB)/
